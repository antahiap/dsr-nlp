This is  Material Type 263. It is based on the anisotropic y ield function proposed by Lou 
and Yoon (Lou and Yoon, 2017).  This yield function  extends the original Drucker 
function into an anisotropic form using a fourth or der linear transformation tensor.  The 
non-associated ﬂow rule (non-AFR) can be applied to  accurately describe both the 
directional yield stresses and |u1D44J-values.  The anisotropic ﬂexibility of this model can be 
further improved by summing up components of the an isotropic Drucker function.  See 
the section Constitutive relations  below for more details.  This model is supported f or 
shell and solid elements.

Hardening laws:

1.  The Swift hardening law 
2.  The Voce hardening law 
3.  The Gosh hardening law 
4.  The Hocket-Sherby hardening law 
5.  A loading curve, where the yield stress is given a s a function of the eﬀective 
plastic strain
